    РЕЖИМЫ РАБОТЫ

    Для любого действия в 3D моделирование существует определенные режимы работы, которые включают в себя интерфейс и дополнительные функции. Режимы упрощают работу и позволяют разделить процесс создания модели поэтапно.

    Режимы blender:

    1. Object Mode


    2. Edit Mode


    3. Sculpt Mode


    4. Vertex Paint mode


    5. Weight Paint mode


    6. Texture Paint mode





    Object Mode (Режим Объекта):

    Назначение: Основной режим для управления целыми объектами.
    Функции:

    • Выделение, перемещение, вращение, масштабирование, дублирование и удаление объектов.
    • Применение модификаторов к объектам.
    • Переключение между объектами.
    Когда использовать: когда нужно работать с объектами как с единым целым, менять их положение в сцене, а не редактировать их геометрию.
    *Переключение в режим редактирования (Edit Mode) выполняется клавишей Tab клавиатуры. Повторное нажатие Tab вернёт в предыдущий режим. Также переходить в другой режим изменения объекта можно с помощью выпадающего списка в заголовке редактора.


    Edit Mode (Режим Редактирования):

    Назначение: режим для изменения геометрии (формы) объектов, например, мешей.
    Функции:

    • Выделение и перемещение, вращение, масштабирование отдельных вершин, рёбер и граней (полигонов).
    • Экструдирование, вырезание, добавление и удаление геометрии.
    • Различные инструменты для моделирования.
    Когда использовать: когда нужно изменить форму объекта, добавить или удалить полигоны, создать сложные модели из простых примитивов.
    *Для перехода в режим редактирования нужно нажать клавишу Tab или выбрать режим Edit Mode в заголовке окна 3D-вида.


    Sculpt Mode (Режим Скульптинга):

    Назначение: режим для моделирования “лепкой”, как если бы вы работали с глиной.
    Функции:

    • Различные кисти для добавления, сглаживания, выдавливания и других деформаций на поверхности модели.
    • Работа с динамической сеткой (Dyntopo) для добавления детализации.
    • Создание органических форм.
    Когда использовать: когда нужно создать сложные органические модели, персонажей, скульптуры, а также добавить детализацию к уже существующим моделям.


    Vertex Paint Mode (Режим Покраски Вершин):

    Назначение: : режим для покраски вершин объекта, позволяющий раскрасить объект без использования текстур (удобно для низкополигональных моделей).
    Функции:

    • Разные кисти для окраски вершин, их смешивания, размытия.
    • Раскраска в зависимости от освещения и других параметров.
    Когда использовать: когда нужно раскрасить объект, особенно простые низкополигональные модели, или добавить детализацию цвета без использования сложных текстур.>


    Weight Paint Mode (Режим Рисования Весов):

    Назначение: режим для определения влияния костей скелета на деформацию меша при анимации.
    Функции:

    • Кисти для раскрашивания вершин, где красный цвет означает максимальное влияние, а синий — минимальное.
    • Настройка плавности влияния костей.
    Когда использовать: когда нужно анимировать модель, деформируя ее поверхность (например, сгибая руку).


    Texture Paint Mode (Режим Текстурного Рисования):

    Назначение: Режим для непосредственного рисования текстур на поверхности объекта.
    Функции:

    • Различные кисти для рисования, штампы, заливки.
    • Возможность использовать текстуры для кистей.
    • Рисование на UV-развертке модели.
    Когда использовать: когда нужно создать уникальные текстуры для объекта, нарисовав их прямо на модели.


    Ключевые моменты:

    • Зависимость: не все режимы доступны для всех типов объектов. Например, Edit Mode работает только с мешами.
    • Контекст: Режим работы определяет, какие инструменты доступны и как они себя ведут.
    • Освоение: чтобы по-настоящему освоить Blender, нужно понимать, какой режим использовать в какой ситуации.





    МОДИФИКАТОРЫ

    Важной частью 3D моделирвания является работа с модификаторами. Они придают необходимые свойства для объекта, к примеру: закругляют модель, добавляют тени и т.д.
    Всего в blendere 54 модификатора, которые делятся на 5 категорий, по принципу их работы

    Категории модификаторы:

    1. Edit


    2. Generate


    3. Deform


    4. Normals


    5. Physics



    Модификаторы Edit (Редактирование):

    Назначение: прямое изменение геометрии, топологии или формы объекта.

    В модификаторы Edit входят:

    1. Data Transfer


    2. Mesh Cache


    3. Mesh Sequence Cache


    4. UV Project


    5. UV Warp


    6. Vertex Weight Edit


    7. Vertex Weight Mix


    8. Vertex Weight Proximity




    Data Transfer (Перенос данных)

    Назначение:
    Копирование данных (веса вершин, UV-координаты, цвета вершин, пользовательские данные и т.д.) с одного объекта на другой. Используется для переноса сложных настроек и атрибутов между мешами, экономя время и усилия.
    Функция:
    Выбирает данные с объекта-источника (source) и применяет их к объекту-приемнику (target). Можно настроить, какие именно данные нужно переносить и как они будут влиять на целевой объект.

    Mesh Cache (Кэш сетки)

    Назначение:
    Запись и воспроизведение деформации меша во времени. Используется для сохранения сложных деформаций, например, от симуляции, и дальнейшего использования на других объектах или в других сценах.
    Функция:
    Копирует выбранные данные с объекта-источника на объект-приемник, модифицируя его геометрию. Позволяет переносить сложные атрибуты и создавать эффекты, которые трудно достичь другими способами.

    Mesh Sequence Cache (Кэш последовательности сетки)

    Назначение:
    Работа с последовательностью изменяющихся сеток, например, результатом симуляции.
    Функция:
    Аналогичен Mesh Cache, но предназначен для работы с последовательностью деформирующихся мешей. Применяет анимированные изменения к геометрии на основе кэшированной последовательности. Позволяет кэшировать сложные анимации и использовать их повторно.

    UV Project (UV-проецирование)

    Назначение:
    Проецирование UV-координат на объект с помощью другого объекта, чаще всего, лампы или камеры. Позволяет создавать сложные UV-развертки на объектах с помощью проецирования
    Функция:
    Выбирает источник проецирования (объект или камеру), устанавливает направление проецирования и применяет UV-координаты к объекту. Позволяет наносить текстуры на поверхности сложной формы.

    UV Warp (UV-искривление)

    Назначение:
    Деформация UV-координат объекта, следуя за деформацией другого объекта. Позволяет создавать сложные эффекты текстурирования, синхронизируя текстуру с изменением объекта.
    Функция:
    Выбирает объект-источник деформации и деформирует UV-координаты целевого объекта в соответствии с ним. Позволяет анимировать текстуру, деформируя ее синхронно с мешем.

    Vertex Weight Edit (Редактирование весов вершин)

    Назначение:
    Точная настройка весов вершин для контроля деформации объектов, например, при использовании арматуры. Используется для создания плавных и реалистичных деформаций.
    Функция:
    Позволяет вручную редактировать веса вершин, используя инструменты выделения и рисования. Позволяет точно настроить, как конкретная вершина влияет на деформацию объекта.

    Vertex Weight Mix (Смешивание весов вершин)

    Назначение:
    Создание более сложных деформаций путем смешивания весов вершин из нескольких групп. Позволяет комбинировать разные группы вершин для более сложного контроля деформаций.
    Функция:
    Выбирает две группы вершин и смешивает их веса с заданной пропорцией. Позволяет создавать плавные переходы между разными деформациями.

    Vertex Weight Proximity (Близость весов вершин)

    Назначение:
    Назначение весов вершинам в зависимости от их близости к другим объектам. Используется для создания динамических эффектов деформации, где веса меняются в зависимости от положения других объектов.
    Функция:
    Выбирает объекты, от которых зависит близость вершин, и задает, как эта близость влияет на веса вершин. Позволяет создавать динамически изменяющиеся веса, основанные на окружении.

    Модификаторы Generate (Генерация):

    Назначение: создания новой геометрии, изменения структуры геометрии или генерации новых форм на основе исходного объекта. Они не просто деформируют существующую геометрию, а скорее используют её как основу для создания чего-то нового или для изменения её топологии.

    В модификаторы Generate входят:

    1. Array

    2. Bevel


    3. Boolean

    4. Build


    5. Decimate

    6. Edge Split


    7. Geometry Nodes

    8. Mask


    9. Mesh to Volume

    10. Mirror


    11. Multiresolution

    12. Remesh


    13. Screw

    14. Skin


    15. Solidify

    16. Subdivision Surface


    17. Triangulate

    18. Volume to Mesh


    19. Weld

    20. Wireframe




    Array (Массив)

    Назначение:
    Создание множества копий объекта, расположенных в массиве. Позволяет быстро и эффективно создавать повторяющиеся структуры.
    Функция:
    Создает копии объекта вдоль одной или нескольких осей, а также по кривой. Можно настраивать смещение, количество копий, и способы трансформации каждой копии.

    Bevel (Скос)

    Назначение:
    Создание фасок или скруглений на острых краях и углах объекта. Используется для придания объектам более гладкого и реалистичного вида.
    Функция:
    Добавляет скошенные или скругленные края к геометрии, уменьшая резкие переходы. Можно настраивать ширину, количество сегментов, и профиль скоса.

    Boolean (Булевы операции)

    Назначение:
    Выполнение булевых операций (объединение, вычитание, пересечение) между двумя объектами. Позволяет создавать сложные формы путем комбинации более простых объектов.
    Функция:
    Выполняет выбранную булеву операцию (разность, объединение, пересечение) между текущим объектом и выбранным объектом-целью.

    Build (Построение)

    Назначение:
    Анимирование появления объекта, создавая эффект “построения” из отдельных частей.
    Функция:
    Анимирует появление объекта, как если бы он собирался по частям. Можно настроить направление, скорость и порядок появления.

    Decimate (Декремация)

    Назначение:
    Уменьшение количества полигонов (треугольников) в объекте. Используется для оптимизации объектов, снижения нагрузки на компьютер и упрощения топологии.
    Функция:
    Уменьшает количество полигонов меша, сохраняя общую форму. Можно настраивать степень децимации и способ её применения.

    Edge Split (Разделение Края)

    Назначение:
    Разделение геометрии по острым углам или по весам. Используется для управления затенением и сглаживанием на различных частях объекта.
    Функция:
    Разделяет вершины вдоль острых углов или по весам, позволяя применить к разным частям разные шейдеры и сглаживания.

    Geometry Nodes (Геометрические ноды)

    Назначение:
    Мощный инструмент для генерации, деформации и модификации геометрии с использованием нодовой системы.
    Функция:
    Выполняет операции с геометрией, используя нодовую систему. Позволяет создавать процедурные эффекты, сложные формы и автоматизировать процессы моделирования.

    Mask (Маска)

    Назначение:
    Скрытие или отображение частей объекта в зависимости от других объектов, групп вершин или весов.
    Функция:
    Скрывает или отображает части геометрии на основе маски, которую можно создать с помощью групп вершин, объектов или весов.

    Mesh to Volume (Преобразование в объём)

    Назначение:
    Преобразование меш-объекта в объёмный. Позволяет использовать меш в качестве основы для объёмных эффектов.
    Функция:
    Преобразует меш в объём, сохраняя его форму. Может использоваться для объёмного рендеринга, создания облаков, тумана и т.д.

    Mirror (Зеркало)

    Назначение:
    Создание зеркальной копии объекта вдоль одной или нескольких осей. Используется для быстрого создания симметричных моделей.
    Функция:
    Создаёт зеркальную копию объекта по указанным осям. Можно настроить смещение, точки поворота и параметры отражения.

    Multiresolution (Мультиразрешение)

    Назначение:
    Создание нескольких уровней детализации меша для скульптинга.
    Функция:
    Позволяет создавать несколько уровней детализации меша, что очень удобно для скульптинга. Чем выше уровень, тем больше детализация.

    Remesh (Пересетка)

    Назначение:
    Перестраивает топологию меша, делая ее более однородной, менее сложной или более подходящей для скульптинга.
    Функция:
    Перестраивает полигоны меша, изменяя их количество и размер. Может упростить сложные меши или сделать их более подходящими для определенных задач, например для скульптинга.

    Screw (Винт)

    Назначение:
    Создание объекта путем вращения профиля вдоль оси. Используется для создания винтообразных и спиралевидных форм.
    Функция:
    Вращает 2D-профиль вокруг оси, создавая трехмерную форму. Можно настроить количество оборотов, угол и другие параметры.

    Skin Modifier (Кожа)

    Назначение:
    Создание объемной геометрии на основе скелета (вершин, соединенных ребрами). Используется для создания органических форм и скелетного моделирования.
    Функция:
    Создает трехмерную геометрию на основе скелета из вершин и ребер. Позволяет управлять толщиной и формой геометрии.

    Solidify (Утолщение)

    Назначение:
    Добавление толщины к поверхности объекта. Используется для создания «оболочки» объекта или для создания толщины плоских поверхностей.
    Функция:
    Создает «оболочку» вокруг объекта, придавая ему толщину. Можно настроить толщину, смещение и другие параметры.

    Subdivision Surface (Разделение поверхности)

    Назначение:
    Увеличение детализации и сглаживание объекта путем разделения его поверхностей на более мелкие.
    Функция:
    Делит поверхность объекта на более мелкие полигоны, добавляя сглаживание. Можно настроить количество разделений.

    Triangulate (Триангуляция)

    Назначение:
    Преобразование всех полигонов объекта в треугольники.
    Функция:
    Разбивает все полигоны меша на треугольники. Полезно при экспорте в другие программы, где треугольники могут быть предпочтительнее.

    Volume to Mesh (Преобразование в меш)

    Назначение:
    Преобразование объемного объекта в меш. Позволяет получить геометрическую форму на основе объема.
    Функция:
    Преобразует объём в меш, используя различные методы, такие как Marching Cubes. Может использоваться для создания различных форм и поверхностей.

    Weld (Сваривание)

    Назначение:
    Объединение вершин, расположенных близко друг к другу. Используется для упрощения и очистки геометрии.
    Функция:
    Соединяет вершины, которые находятся в пределах заданной дистанции, удаляя лишнюю геометрию и делая меш более однородным.

    Wireframe (Каркас)

    Назначение:
    Создание каркасной структуры из ребер объекта.
    Функция:
    Создает каркас из ребер объекта, позволяя сделать его «ребристым». Можно настраивать толщину и другие параметры.

    Модификаторы Deform:

    Назначение: изменения формы объектов путем деформации их геометрии. Они позволяют создавать различные виды искривлений, растяжений, сжатий и других деформаций. Главное отличие от “Generate” - они не создают новую геометрию, а модифицируют существующую.

    В модификаторы Deform входят:

    1. Armature

    2. Cast


    3. Curve

    4. Displace


    5. Hook

    6. Laplacian Deform


    7. Lattice

    8. Mesh Deform


    9. Shrinkwrap

    10. Simple Deform


    11. Smooth

    12. Smooth Corrective


    13. Smooth Laplacian

    14. Surface Deform


    15. Volume Displace

    16. Warp


    17. Wave



    Armature (Арматура)

    Назначение:
    Привязка объекта к скелету (арматуре) для анимации и деформации.
    Функция:
    Привязывает объекты к костям арматуры, позволяя управлять их деформацией с помощью настроек на костях.

    Cast (Отливка)

    Назначение:
    Изменение формы объекта, придавая ему цилиндрическую, сферическую или кубическую форму.
    Функция:
    Деформирует объект, используя заданные параметры для создания форм, напоминающих цилиндр, сферу или куб.

    Curve (Кривая)

    Назначение:
    Деформация объекта вдоль кривой.
    Функция:
    Деформирует объект, следуя за формой кривой. Используется для создания различных изгибов, скручиваний и других деформаций.

    Displace (Смещение)

    Назначение:
    Смещение вершин объекта в соответствии с текстурой.
    Функция:
    Использует текстуру для смещения вершин объекта, создавая рельеф и детализацию на основе ее данных.

    Hook (Крюк)

    Назначение:
    Привязка вершин объекта к другим объектам для управления деформацией.
    Функция:
    Привязывает вершины объекта к другим объектам, позволяя изменять их положение, используя деформации целевого объекта.

    Laplacian Deform (Деформация Лапласиана)

    Назначение:
    Деформация объекта с сохранением объема и формы.
    Функция:
    Деформирует объект, сохраняя его объем и форму, используя математические принципы Лапласиана.

    Lattice (Решетка)

    Назначение:
    Деформация объекта с помощью решетки (сетки) контрольных точек.
    Функция:
    Деформирует объект, используя решетку, управляющую перемещением вершин объекта.

    Mesh Deform (Деформация сетки)

    Назначение:
    Деформация объекта с помощью другого меша.
    Функция:
    Деформирует объект в соответствии с формой другого меша.

    Shrinkwrap (Обёртывание)

    Назначение:
    Прикрепление объекта к поверхности другого объекта.
    Функция:
    Деформирует объект, прикрепляя его к поверхности другого объекта.

    Simple Deform (Простая деформация)

    Назначение:
    Простое искривление объекта.
    Функция:

    Сглаживание геометрии объекта.
    Функция:

    Smooth Corrective (Корректирующее сглаживание)

    Назначение:
    Корректирующее сглаживание геометрии.
    Функция:
    Сглаживает геометрию, улучшая гладкость поверхности, без удаления существенных деталей.

    Smooth Laplacian (Корректирующее Лапласиана)

    Функция:

    Искривление объекта вокруг заданной точки.
    Функция:

    Назначение:
    Добавление волновой деформации к объекту.
    Функция:

    Модификаторы Normals:

    Назначение: для управления тем, как объект взаимодействует со светом при рендеринге. Они изменяют направление нормалей, которые, в свою очередь, влияют на затенение, отражения и общее восприятие поверхности. Они не изменяют геометрию объекта напрямую, но влияют на её визуальное представление.

    В модификаторы Normals входят:

    1. Normal Edit


    2. Weighted Normal


    3. Smooth by Angle



    Normal Edit Modifier (Редактирование нормалей)

    Назначение:
    Непосредственное редактирование нормалей объекта для управления затенением и видимостью поверхностей.
    Функция:
    Позволяет изменять направление нормалей вершин, полигонов или ребер. Это влияет на то, как объект будет выглядеть при рендеринге, позволяя исправлять артефакты затенения, создавать эффект фальшивых отражений или другие специальные эффекты. Можно выбирать нормали и редактировать их направление, угол наклона или среднее значение для группы.

    Weighted Normal Modifier (Взвешенные нормали)

    Назначение:
    Улучшение затенения объектов с острыми краями или плоскими поверхностями.
    Функция:
    Вычисляет и корректирует нормали, основываясь на площади поверхности полигонов, что предотвращает появление артефактов затенения и делает поверхность более гладкой визуально. Можно использовать “веса” для влияния на коррекцию нормалей. Помогает сгладить переходы и скрыть недостатки модели при рендеринге.

    Smooth by Angle Modifier (Сглаживание по углу)

    Назначение:
    Сглаживание нормалей объекта, основываясь на угле между полигонами.
    Функция:
    Вычисляет и сглаживает нормали на тех участках объекта, где угол между полигонами меньше заданного значения. Помогает сгладить поверхность без использования разделения полигонов, как при Subdivision Surface. Применяется на гранях и ребрах, сглаживая переходы и уменьшая резкие углы.

    Модификаторы Physics:

    Назначение: позволяют создавать физические симуляции в Blender, имитируя поведение объектов в реальном мире, таких как: ткань, жидкости, твердые тела, частицы. Они используются для создания реалистичных анимаций и эффектов.

    В модификаторы Physics входят:

    1. Cloth


    2. Collision


    3. Dynamic Paint


    4. Explode


    5. Fluid


    6. Ocean


    7. Particle Instance


    8. Particle System


    9. Soft Body



    Cloth (Ткань)

    Назначение:
    Симуляция ткани, имитируя её поведение под воздействием внешних сил.
    Функция:
    Позволяет объекту вести себя как ткань, реагируя на гравитацию, ветер и столкновения с другими объектами.

    Collision (Столкновение)

    Назначение:
    Определение объекта как объекта столкновения для других физических симуляций.
    Функция:
    Позволяет другим симуляциям (например, ткани или мягких тел) взаимодействовать с объектом и избегать проникновений.

    Dynamic Paint (Динамическая краска)

    Назначение:
    Создание динамических красок на поверхности объекта, взаимодействующих с другими объектами.
    Функция:
    Позволяет объектам оставлять "следы" краски при контакте с другими объектами, создавая интерактивные эффекты.

    Explode (Взрыв)

    Назначение:
    Имитация взрыва объекта.
    Функция:
    Разбрасывает части объекта в разных направлениях, создавая эффект взрыва.

    Fluid (Жидкость)

    Назначение:
    Симуляция жидкостей и газов.
    Функция:
    Позволяет создавать реалистичные симуляции жидкостей и газов, взаимодействующих с другими объектами и окружением.

    Ocean (Океан)

    Назначение:
    Симуляция поверхности океана.
    Функция:
    Генерирует реалистичную поверхность океана, которая может анимироваться для создания эффекта волн.

    Particle Instance (Инстанс частиц)

    Назначение:
    Использование частиц для инстансирования объектов.
    Функция:
    Позволяет отображать объекты вместо частиц, создавая эффекты размножения объектов или распределения по поверхности.

    Particle System (Система частиц)

    Назначение:
    Создание и управление системами частиц.
    Функция:
    Позволяет создавать системы частиц для имитации эффектов, таких как дождь, дым, снег и т.д. Можно управлять их свойствами и движением.

    Soft Body (Мягкое тело)

    Назначение:
    Симуляция поведения мягких тел (например, резины, желе).
    Функция:
    Позволяет объектам вести себя как мягкие тела, деформируясь при воздействии внешних сил или столкновениях.
